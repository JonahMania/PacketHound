# Packet Hound

### A tool for capturing and viewing packet data

## Install

Install libpcap-dev
* Ubuntu - sudo apt-get install libpcap-dev
* Arch - sudo pacman -Ss libpcap

`$ npm install`


## Usage

### Routes
* /packets.json
* /packet.json?id=[id]
* /session.json
* /database.json
* /interfaces.json
* /packets
* /packet?id=[id]
